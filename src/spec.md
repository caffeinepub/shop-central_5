# Specification

## Summary
**Goal:** Update Shop Central to use Indian Rupees (INR) as currency, add Food category, and include delivery type options for products.

**Planned changes:**
- Change all price displays and inputs from USD ($) to Indian Rupees (₹/INR) across product cards, details, cart, checkout, and order confirmation
- Add 'Food' as a new category option in product creation form and catalog navigation filters
- Add delivery type field with two options: '1-hour delivery' and 'Takeaway' to products, visible in creation form, product cards, and detail pages

**User-visible outcome:** Users can create and browse food products with prices displayed in INR, and see delivery options (1-hour delivery or takeaway) for each product.
